# kvKit.swift

A collection of general purpose auxiliaries on Swift. For example:

- Specific collections.
- Task dispatching.
- Mathematical auxiliaries.
- UI.
- Bonjour.
- Compression.
- WebKit tasks.


## Supported Platforms

This package contains both cross-platform code and platform specific code.


## Getting Started

#### Package Dependencies:

```swift
.package(url: "https://github.com/keyvariable/kvKit.swift.git", from: "4.2.0")
```
#### Target Dependencies:
```swift
.product(name: "kvKit", package: "kvKit.swift")
```
#### Import:
```swift
import kvKit
```
